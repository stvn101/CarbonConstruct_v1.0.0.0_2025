=========================

  POST   /api/v1/calculate/fuel        Calculate fuel emissions
  POST   /api/v1/calculate/material    Calculate material carbon
  POST   /api/v1/calculate/waste       Calculate waste emissions
  GET    /api/v1/calculate/materials   List available materials
  GET    /api/v1/calculate/fuels       List available fuels

  POST   /api/v1/projects              Create project
  GET    /api/v1/projects              List projects
  GET    /api/v1/projects/{id}         Get project details

Database Tables (4 main + audit log)
=====================================

nger_operational_factors  (63 rows)
  - scope, category, fuel_type, region, unit
  - co2_factor, ch4_factor, n2o_factor, total_co2e
  - Covers: Diesel (stat/trans), Petrol, Gas, Electricity by state

nger_materials  (123 rows)
  - material_type, category, unit, data_quality
  - a1a3_default/min/max/avg_per_unit
  - carbon_storage_per_unit (negative for timber)
  - Covers: Concrete, Steel, Timber, Glass, Insulation

projects
  - project_id, project_name, postcode, state, ncc_volume

calculation_log  (audit trail)
  - Every calculation with timestamp, factor_source, result


Quick Start
===========

Backend:
  cd backend
  ./DEPLOY.sh
  source venv/bin/activate
  uvicorn api.main:app --reload --port 8000

Frontend:
  cd frontend
  # Copy files to your Vite app
  cp -r src/lib/api ~/your-app/src/lib/
  cp -r src/components/calculator ~/your-app/src/components/

  # Add to .env
  echo "VITE_API_URL=http://localhost:8000/api/v1" >> .env

  # Run your app
  npm run dev


Test It:
  # API docs
  http://localhost:8000/api/docs

  # Health check
  curl http://localhost:8000/api/health

  # Calculate diesel emissions
  curl -X POST http://localhost:8000/api/v1/calculate/fuel \
    -H "Content-Type: application/json" \
    -d '{
      "project_id": "TEST001",
      "fuel_type": "Diesel",
      "quantity": 5000,
      "unit": "L",
      "state": "NSW",
      "is_stationary": false
    }'
